What this page does ?

Takes the number between 144 and 50458 because model takes 144 previous steps to predict the next 72 steps, therefore the minimum number should be 144, then it takes that number fetches previous 144 data points from the dataset and predicts the best time for which Active power will be high for next 72 hours.




Due to the unavailability of any API which provides the data of about wind speed, wind gust, wind direction,etc. of the wind farm in real time, also these are some of the features used for building the model.

Therefore, for the evaluation, the application uses the dataset only used for building the model.

Also according to the problem statement, the application should predict the wind power for the next 72 hours, which requires alot of training, therefore due to the lack of computation power and TIME, the model is not trained well, it should be trained for many epochs.
